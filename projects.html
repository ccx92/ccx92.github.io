Computer Science Honours Project

Comparing ASP.NET and Web Builder Development Methods

This project was to compare web builder software against the .NET Framework to discover key differences in web development methods. This was done by performing research into existing information on both to gain a ground basis of the main differences before planning and building websites using both approaches.

For the practical comparison, I attempted to build a blog website based on a simple design using a web builder (WiX) and a website close as possible using C#, SQL in Visual Studio.

Advanced Programming for Mobile Devices

Mobile Companion Application

A project for my Advanced Mobile Programming module in university, my goal was to build a companion web application for an online MMORPG which would act as a convenient go-to place for information. The application held several webpages designed in a uniformed style to show relevance to the MMORPG.

The application was only optimised for use on mobiles and thus isn't designed for use on a desktop/laptop screen. Besides the HTML and CSS used to customise the layout, it also includes some limited JavaScript functionality used to enable the Firebase features.

Collabrative Virtual Environments

Multiplayer Virtual World

A project for my Collabrative Virtual Environments module in university, my goal was to build a virtual software house environment in a group project scenario. The world allowed a number of players to join a server in which all were assigned an avatar that could explore and interact with the world.

The application was built using the Unity Engine development environment using C#. Utilising Unity Collab to synchronise our work, the group was able to actively construct the world using assets in real time together using the Scene Fusion software. A demo video is available of the project at the end of the linked documentation.
